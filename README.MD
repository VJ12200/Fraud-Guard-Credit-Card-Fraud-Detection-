# FraudGuard — Real-Time Credit Card Fraud Detection


## Overview
Two-layer fraud detection system combining a Java rule engine
with an XGBoost ML model trained on 1.85M transactions.
PR-AUC: 0.967 | F1: 0.922 | Recall: 98.7%


# Data

Download `fraudTrain.csv` from:
https://www.kaggle.com/datasets/kartik2112/fraud-detection

Place it in this directory before running train.py.
